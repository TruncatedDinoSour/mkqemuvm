# Mkqemuvm

> A simple wrapper around qemu to make virtual machines

# Requirements

- mlocalte
- fzf
- qemu
- edk2-ovmf

# Installation

## Manual

```bash
sudo install -Dm755 mkqemuvm /usr/local/bin
```

## Packages

- Linux
  - Gentoo linux: [app-emulation/mkqemuvm::dinolay](https://ari-web.xyz/gentooatom/app-emulation/mkqemuvm)
